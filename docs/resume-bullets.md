# Resume Bullets for OpsFlow AI

## SDE / SWE Version

- Built and deployed OpsFlow AI, a full-stack workflow automation platform using Next.js, FastAPI, PostgreSQL/pgvector, Redis, Docker, and JWT auth for ops-ticket ingestion and human-in-the-loop action approval.
- Designed normalized schemas for work items, data sources, suggested actions, and audit logs; implemented CSV ingestion, rule-based AI classification, action generation, approval/rejection APIs, and metrics aggregation.
- Added Docker Compose infra, backend tests, GitHub Actions CI, architecture docs, demo credentials, and sample data to make the project recruiter-demo ready.

## FDE / AI Product Version

- Developed an FDE-style AI workflow automation platform that converts messy support data into actionable workflows, generating draft replies, escalations, and internal tasks with explainability and human approval.
- Implemented trust and reliability features including audit logs, confidence scores, action payload previews, approval-rate metrics, failed-action tracking, and deterministic offline AI fallback for demos.
- Built a recruiter-friendly live demo with sample ops dataset, dashboard metrics, approval workflow, and a 90-second demo script to communicate product and engineering impact.
